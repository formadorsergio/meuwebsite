Tech Tips

    |1| Formatando um disco no Windows com o Diskpart!

  • list disk - Comando para listar os discos físicos no pc (PT). | Command used to check physical disks on pc (EN).
  • select disk - Comando para escolher o disco (PT). | Command to choose the disk (EN).
  • clean - Comando para limpar o disco na totalidade (PT). | Command to wipe out all disk (EN).
  • create partition primary - Comando para criar uma partição regular (PT). | Command to create a common partition (EN).
  • format fs=ntfs quick - Comando para formatar rapidamente o volume no sistema de ficheiros NTFS (PT). | Command to format the volume with NTFS file system (EN).
